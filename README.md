Rails Q&A software.

## Installation
The usual sort of thing:

 * Clone the repo and `cd` into the directory
 * Fill in `config/database.yml` with the correct values for your environment
 * Run `rails db:setup` and `rails db:migrate`
 * Run `rails s`
